# polyfill-base64
A polyfill used to add btoa and atob to the global scope.

## 🚀 Getting Started

Using [`yarn`](https://yarnpkg.com):

```bash
yarn add @react-native-anywhere/polyfill-base64
```

## ✍️ Usage

```javascript
import "@react-native-anywhere/polyfill-base64";

// btoa, atob...
```
